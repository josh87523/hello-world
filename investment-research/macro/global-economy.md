## 关键要点

> **核心判断：全球经济在分化中保持韧性，但下行风险犹存。**

- **全球增长稳健但分化加剧：** IMF预测2026年全球GDP增长3.3%，较2025年10月预测上调。美国经济表现超预期（2.4%），印度维持高速增长（6.3%），而日本仅增长0.6%，欧元区增长1.2%。中国经济增速预计放缓至4.5%-4.8%。

- **美联储进入观望期：** 联邦基金利率维持在3.5%-3.75%区间不变，结束此前连续三次降息。内部官员对未来路径存在明显分歧，部分官员甚至提出加息可能性。市场预计2026年最多降息两次，鲍威尔主席任期将于5月届满，继任者Kevin Warsh的提名增加了政策不确定性。

- **通胀回落但仍高于目标：** 美国1月CPI同比2.4%，核心CPI降至2.5%（2021年4月以来最低）。欧元区通胀降至1.7%。全球通胀预计从2025年的4.1%降至2026年的3.8%。但关税传导效应和劳动力市场动态构成上行风险。

- **美债收益率温和回落：** 10年期国债收益率在4.05%附近，较一年前下降22个基点。AI冲击担忧和贸易政策不确定性推动资金流入固定收益市场。

- **美元显著走弱：** DXY指数跌至97.5附近，过去12个月下跌9.06%，一度触及5年低点。美元走弱利好新兴市场资产。

- **最高法院推翻IEEPA关税成为最大变量：** 2月20日，美国最高法院裁定总统无权依据IEEPA征收关税，导致此前已征收的1600亿美元以上关税面临退还争议。特朗普随即援引《1974年贸易法》第122条实施10%全球关税（后提至15%），有效期150天。

- **供应链进入"结构性波动"时代：** 地缘政治碎片化（97%威胁等级）、极端天气加剧（93%威胁等级）、基础设施老化和网络安全威胁构成四大风险。东南亚替代产能持续增长，"中国+1"策略加速演进。

- **劳动力市场韧性犹存：** 美国1月新增非农就业13万人，失业率小幅降至4.3%。但2025年全年非农就业增长被大幅下修至18.1万人（此前报告58.4万人），暗示劳动力市场实际表现弱于前期估计。

## 一、全球GDP增长展望

### 1.1 IMF世界经济展望（2026年1月更新）

国际货币基金组织（IMF）2026年1月发布的《世界经济展望》更新预测全球经济增长率为 **3.3%**（2026年）和 **3.2%**（2027年），较2025年10月预测略有上调。

IMF指出，表面上稳定的增长数据掩盖了深层分化：**贸易政策转向带来的逆风**被**AI相关投资激增的顺风**所抵消，后者在北美和亚洲尤为显著。此外，宽松的财政和货币政策、宽裕的金融条件以及私营部门的适应能力也提供了支撑。

| 经济体 | 2026年GDP增长预测 | 较上次修正 |
|--------|-------------------|------------|
| 全球 | 3.3% | +0.1ppt |
| 美国 | 2.4% | +0.3ppt |
| 欧元区 | 1.2% | +0.2ppt |
| 中国 | 4.5% | 持平 |
| 印度 | 6.3% | 持平 |
| 日本 | 0.6% | -0.1ppt |
| 中东与中亚 | 3.9% | +0.2ppt |

### 1.2 世界银行全球经济展望（2026年1月）

世界银行将2026年全球增长预测上调至 **2.6%**（采用市场汇率权重，与IMF的购买力平价权重方法不同），较此前的2.4%有所提高。世界银行指出，全球经济在面对"历史性"贸易紧张局势升级时表现出了"令人惊讶的抗冲击能力"。

**区域展望：**

| 地区 | 2026年增长预测 | 关键驱动因素 |
|------|---------------|-------------|
| 东亚与太平洋 | 4.4% | 出口韧性，但增速放缓 |
| 南亚 | 6.2% | 贸易限制影响增速 |
| 欧洲与中亚 | 2.4% | 内需稳固抵消欧元区疲软 |
| 拉美与加勒比 | 2.3% | 贸易紧张和内需疲软 |
| 中东与北非 | 3.6% | 石油出口国活动扩张 |
| 低收入国家 | 5.7% | 增长加速 |

### 1.3 中国经济展望

中国2025年GDP增速达到5%，符合政府目标。但2026年增速预计放缓：

| 机构 | 2026年中国GDP增长预测 |
|------|---------------------|
| 高盛 | 4.8%（最乐观） |
| 花旗 | 4.7% |
| IMF | 4.5% |
| 瑞银 | 4.5% |
| 先锋领航 | 4.5% |
| 世界银行 | 4.4% |
| 荣鼎集团 | <3%（最悲观，质疑官方数据） |

**核心挑战：** 房地产深度调整（新开工面积较峰值下降75%，房地产投资下降50%）、消费者信心接近疫情低点、K型增长格局（新经济强劲 vs. 传统经济疲弱）。

**积极因素：** 出口韧性（2025年实际出口增长约8%）、政策支持（预计约1万亿元额外财政刺激、20个基点降息、50个基点降准）、AI和新能源等新经济增长点。2026年是"十五五"规划开局之年，政策定调尤为关键。

### 1.4 主要风险

IMF和世界银行均指出下行风险包括：
- 贸易摩擦重新升级和政策不确定性
- 全球金融条件收紧
- 财政脆弱性上升
- 地缘政治紧张局势和冲突
- 气候和公共卫生冲击
- AI投资驱动的增长基础过于狭窄

## 二、美联储货币政策与利率走势

### 2.1 2026年1月FOMC会议决议

美联储于2026年1月28日维持联邦基金利率在 **3.50%-3.75%** 区间不变，结束此前连续三次各25个基点的降息。这一决定符合市场预期。

**FOMC声明要点：**
- "现有指标显示经济活动以稳健步伐扩张"
- "就业增长仍然偏低，失业率出现企稳迹象"
- "通胀仍略高于目标水平"

### 2.2 内部分歧加剧

值得关注的是，本次会议出现两位委员投反对票——Stephen I. Miran和Christopher J. Waller均倾向于降息25个基点。两人均由特朗普总统任命。

2月18日发布的会议纪要显示，美联储内部官员对利率前景分歧明显：
- **鸽派：** 认为在通胀配合的情况下，年内晚些时候可以恢复降息
- **鹰派：** 部分官员"提出了上调利率目标区间的可能性"，认为如果通胀持续高于目标，加息应该被考虑

### 2.3 领导层更迭

美联储主席杰罗姆·鲍威尔的任期将于 **2026年5月届满**。前美联储理事 **Kevin Warsh** 已被提名为下一任主席。Warsh被视为"改革派相对鹰派"，但至今尚未就货币政策发表任何公开声明，引发市场大量猜测。

### 2.4 市场定价与利率路径

| 指标 | 当前数据 |
|------|---------|
| 联邦基金利率 | 3.50%-3.75% |
| 2026年降息预期 | 最多两次 |
| 3月会议预期 | 按兵不动 |
| 6月降息概率 | 约83%（CME FedWatch） |
| 2026年底预计利率 | 3.00%-3.25%区间 |

**核心判断：** 美联储短期内将保持观望，任何降息决策取决于通胀和就业数据的演变。领导层更迭增加了下半年货币政策的不确定性。

- **关税传导效应：** 多家研究机构指出，关税对消费者价格的延迟传导可能在2026年中期推高通胀

## 四、美国国债收益率与债券市场

### 4.1 当前收益率水平

截至2026年2月25日：

| 期限 | 收益率 | 近一个月变化 | 近一年变化 |
|------|--------|-------------|-----------|
| 2年期 | 3.48% | — | — |
| 10年期 | 4.04%-4.05% | -17bps | -22bps |
| 30年期 | 4.72% | — | — |

10年期国债收益率处于近三个月最低水平附近。

### 4.2 关键驱动因素

1. **AI颠覆担忧：** 软件服务和支付领域的估值反映出市场对AI技术可能以不可预测方式取代传统行业的担忧，推动资金从股市转向国债

2. **贸易政策不确定性：** 欧洲和亚洲主要经济体考虑暂停与美国达成的贸易协议，加剧了市场避险情绪

3. **最高法院关税裁决：** IEEPA关税被推翻后，关税收入的损失可能加剧美国财政赤字，引发债券投资者担忧

4. **美联储政策：** 收益率曲线趋于平坦化，市场推后了对美联储降息的预期

5. **财政赤字风险：** 关税收入大幅减少后，联邦财政压力增大

### 4.3 收益率曲线形态

2年期/10年期利差约为 **+56bps**，收益率曲线维持正常形态（正斜率），表明市场对短期衰退风险的担忧有所缓解，但长端收益率仍反映出对通胀和财政可持续性的忧虑。

## 六、地缘政治风险

### 6.1 美中关系与贸易战

#### 关税演变时间线

| 时间节点 | 事件 |
|---------|------|
| 2025年4月2日 | 美国"解放日"对华加征34%关税（叠加20%芬太尼关税） |
| 2025年4月 | 关税螺旋升级：美方升至145%，中方反制至125% |
| 2025年5月12日 | 双方达成停火：美方降至30%，中方降至10% |
| 2025年10月30日 | **釜山会晤：** 特朗普与习近平在韩国釜山会面，达成暂停新一轮关税升级的框架协议 |
| 2026年2月20日 | **最高法院裁定IEEPA关税违宪** |
| 2026年2月21日 | 特朗普援引第122条实施10%全球关税，随后提至15% |

#### 最高法院IEEPA裁决——2026年最大制度性变量

2026年2月20日，美国最高法院以 **6:3** 裁定总统无权依据《国际紧急经济权力法》（IEEPA）征收关税，认定关税属于"征税权"，根据宪法第一条属于国会权力。

**核心影响：**
- 此前通过IEEPA征收的关税总额超过 **1,600亿美元**（宾夕法尼亚大学沃顿模型估计为1,750-1,790亿美元）
- IEEPA关税每日征收约 **5亿美元**
- 超过1,000家企业已申请关税退还，数量预计将大幅增长
- 裁决将对华进口关税有效税率降低了约 **三分之二**

**行政应对：** 特朗普立即援引《1974年贸易法》第122条实施替代关税（10%→15%），有效期150天（至2026年7月24日）。但第122条面临进一步法律挑战。

**财政影响：** 财政部长Bessent表示，结合第122条、第232条和第301条关税，"2026年关税收入将基本不变"。但如果第122条关税在150天后到期，2026年平均有效关税率预计为5.6%（1972年以来最高）。

#### 对家庭的影响

特朗普关税相当于2026年每户美国家庭平均 **增加1,500美元税负**。到2025年10月，进口商仅承担27%的关税成本，出口商承担18%，消费者承担 **55%**。

### 6.2 关键矿产与稀土博弈

中国控制全球 **85%** 的稀土加工能力，全球95%的半导体级硅来自仅四家公司。中国不太可能完全放弃其稀土"卡脖子"优势，这些资源将继续成为中美博弈中的重要筹码。

### 6.3 军事热点

| 热点区域 | 专家评估军事升级可能性 |
|---------|---------------------|
| 南海 | 43%的专家认为最可能 |
| 台湾海峡 | 33%的专家认为最可能 |

但多数分析认为，中国的军事胁迫不太可能超越灰色地带活动的范畴。2026年美中关系预计将维持"可控的不稳定"状态——双方合作刚好足以避免崩盘，但战略竞争的本质不变。

### 6.4 其他地缘政治风险

- **美伊核谈判：** 第三轮谈判于2月在日内瓦举行
- **俄乌冲突持续：** 波罗的海地区的俄罗斯无人机入侵和GPS干扰自2024年以来翻倍
- **格陵兰危机：** 2026年初冲击美元信用和盟友关系

### 6.5 企业信心调查

2026年中国美国商会调查显示，**79%** 的受访者对中美关系持中性或积极态度——较上年改善30个百分点。但企业仍高度关注风险管理、供应链多元化和地缘政治风险纳入投资决策。

## 七、全球供应链状况

### 7.1 "结构性波动"时代

世界经济论坛2026年1月报告指出，全球价值链已进入 **"结构性波动"时代**，迫使企业和政府重新评估投资和生产布局。**74%** 的企业领导者现在将韧性投资视为增长驱动力。

### 7.2 四大核心风险

| 排名 | 风险类别 | 威胁等级 | 关键数据 |
|------|---------|---------|---------|
| 1 | 地缘政治碎片化与贸易管制 | **97%** | 关键矿产出口管制2023-2025年翻倍，其他贸易限制增长167% |
| 2 | 极端天气加剧 | **93%** | 2025年欧洲夏季损失430亿欧元；十亿美元级气象灾害现每三周发生一次 |
| 3 | 基础设施老化 | 高 | 麦肯锡估计2040年前需106万亿美元基础设施投资 |
| 4 | 网络安全与混合战争 | 高 | 2026年物流网络攻击预计翻倍 |

### 7.3 关税对贸易流的冲击

2025年，主要经济体间的关税升级导致超过 **4,000亿美元** 的全球贸易流重新配置，主要航运路线中断导致集装箱运费同比上涨40%。

### 7.4 供应链重构——"中国+1"加速

随着中国产能份额减弱，东南亚持续获益：

| 来源国 | 美国进口同比增长 |
|-------|-----------------|
| 泰国 | +33% |
| 印尼 | +34% |
| 越南 | +15.4% |

"中国+1"策略正在演变为更广泛的多区域网络布局。

### 7.5 关键材料短缺风险

- 中国控制 **85%** 的稀土加工
- 全球95%的半导体级硅来自四家公司
- 关键矿产供应中断将影响美国能源、军事和商业应用

### 7.6 监管变化

2026年，欧盟 **碳边境调节机制（CBAM）** 正式生效，要求企业对特定进口商品的碳排放进行核算。

### 7.7 行业应对

供应链管理协会CEO指出："去年是管理中断，而现在是重新设计全球网络。"生成式AI和智能代理系统正加速嵌入供应链平台，实现采购、规划和物流的实时响应。

#### 新兴市场——构建多元化配置的机遇

2. **估值优势：** EM估值相对发达市场具有吸引力
3. **央行宽松：** 新兴市场央行降息空间充裕
4. **AI外溢效应：** 中国的廉价电力优势有利于AI发展
5. **地理多元化收益：** 2025年美国罕见地落后于欧洲、中国和亚洲市场——分散化配置已开始显示其价值

历史数据显示，自1957年以来的17个中期选举周期中，S&P 500有12次进入修正。1950年以来中期选举年的平均最大回撤为 **-17.5%**。
