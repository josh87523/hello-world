### 四、投资策略核心要点

**总体配置**（中高风险偏好）：
```
股票 80% = 美股50% + 港股20% + A股10%
固收 10% = 美债5% + 企业债5%
现金  8% = 备用仓位
黄金  2% = 对冲
```

**风险管理三原则**：
1. 单一持仓不超过总组合 **10%**
2. 个股止损线：买入成本下跌 **20%**
3. 组合整体最大回撤警戒线：**15%**
