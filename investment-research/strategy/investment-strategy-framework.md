## 一、投资理念与核心原则

### 1.1 投资哲学

本框架融合**价值投资**与**成长投资**精髓，核心理念如下：

> **"以合理价格买入优质成长企业，长期持有，让时间复利发挥威力。"**

**五大核心原则**：

1. **质量优先**：只买行业龙头或细分赛道冠军，护城河清晰可辨
2. **成长为本**：优先配置收入/利润增速15%+的企业，优秀商业模式创造超额回报
3. **估值纪律**：再好的公司也不能无限价格买入，PEG < 1.5x 为优选区间
4. **分散对冲**：跨市场、跨行业、跨资产分散，单一持仓不超过总组合10%
5. **动态调整**：宏观环境变化时及时再平衡，不固执于既有判断

### 1.2 与巴菲特/芒格原则的对比调整

| 传统价值投资 | 本框架调整 | 原因 |
|------------|-----------|------|
| 极低PE/PB | 接受合理成长溢价 | AI时代高增速值得溢价 |
| 集中持股 | 适度分散（15-25只） | 科技股波动率高 |
| 长期持有，不动 | 设定止损和再平衡规则 | 控制下行风险 |
| 远离科技股 | 科技是核心配置 | 时代变迁，护城河新形态 |
| 忽视宏观 | 宏观驱动行业轮动 | 利率/汇率影响显著 |

## 三、选股标准体系

### 3.1 定量筛选标准（硬门槛）

**成长性指标**：
- 年营收增速 ≥ 15%（连续3年）
- 净利润增速 ≥ 20%（或正在实现盈利转正）
- 自由现金流正向或明确路径

**盈利质量指标**：
- 毛利率 ≥ 40%（软件/平台型）或 ≥ 20%（硬件/制造型）
- 净利率持续改善趋势
- 经营性现金流/净利润 > 1（真实盈利，非纸面）

**估值指标**：
- PEG < 1.5x（优选 < 1.0x）
- PS < 20x（对于亏损成长股）
- EV/EBITDA < 50x（盈利企业）

**财务健康指标**：
- 资产负债率 < 50%（科技公司）
- 流动比率 > 1.5
- 净现金 > 0 或有明确偿债计划

### 3.2 定性筛选标准（软门槛）

**护城河评估**（评分1-5分，要求≥3分）：
- [ ] 网络效应（用户越多价值越大）
- [ ] 转换成本（用户迁移成本高）
- [ ] 成本优势（规模效应或技术壁垒）
- [ ] 无形资产（品牌、专利、数据）
- [ ] 高效规模（有限市场先发垄断）

**管理层评估**：
- 创始人/核心管理层持股 > 5%（利益绑定）
- 资本配置历史良好（回购、并购ROI高）
- 公开承诺与实际执行一致性

**行业地位评估**：
- 细分市场份额 > 25%（优选龙头）
- 行业结构：寡头竞争优于完全竞争
- TAM（可寻址市场）> $100B，且仍处增长期

### 3.3 淘汰标准（一票否决）

以下任一情况直接排除：
- 财务造假历史或审计意见保留
- 持续依赖外部融资维持运营（烧钱无路径）
- 主营业务被颠覆性技术威胁且无应对策略
- 管理层大量减持（过去12个月 > 5% 持仓）
- 估值严重泡沫化（PEG > 3x 且无特殊催化）

## 四、买入策略

### 4.1 建仓时机选择

**理想买入场景**（优先级从高到低）：

1. **优质股+市场系统性下跌**：大盘跌10%+，优质个股跟跌，最佳加仓时机
2. **个股基本面超预期，股价未反应**：财报超预期但股价未大涨，48小时内买入
3. **行业轮动，暂时性资金撤离**：估值合理，因短期情绪下跌>15%
4. **技术面回调至均线支撑**：基本面不变，股价回调至50/200日均线

**避免买入场景**：
- 追高：股价已涨30%+，脱离均线，RSI > 75
- 抄底未止跌：趋势明显向下，没有技术企稳信号
- 重大不确定事件前：财报前一天、重大监管审查期间

### 4.2 分批建仓纪律

```
建仓策略：3次分批买入
├── 第一批（30%仓位）：初始判断正确，估值合理时买入
├── 第二批（40%仓位）：价格回调5-10%，基本面确认
└── 第三批（30%仓位）：趋势确认，价格开始反弹时

注意：不要在价格下跌过程中过快买满仓
```

### 4.3 不同市场状态下的操作策略

| 市场状态 | 判断标准 | 操作策略 |
|---------|---------|---------|
| 牛市初期 | VIX < 15，市场突破新高 | 积极加仓，偏向成长股 |
| 牛市中期 | 市场稳步上涨，情绪乐观 | 维持仓位，定期再平衡 |
| 牛市末期 | VIX异常低，散户大量入场 | 减仓至60-70%，增加防御 |
| 熊市初期 | 市场跌破200日均线 | 减仓至40-50%，现金为王 |
| 熊市中期 | 市场持续下跌，恐慌蔓延 | 维持低仓位，逢低小幅买入 |
| 熊市末期 | VIX > 40，恐慌性抛售 | 积极布局，大幅加仓优质股 |

## 五、卖出策略

### 5.1 止损规则（纪律性止损）

**绝对止损线**：
- **单只股票**：买入成本下跌 **20%** 触发止损考量
- **组合层面**：净值下跌 **15%** 触发全面风险审查

**止损执行原则**：
- 止损不是认输，是保护本金用于下一次机会
- 止损后等待情绪稳定（最少48小时）再重新分析
- 严禁"摊平"亏损股（基本面未改善时）

**动态止损（Trailing Stop）**：
- 股价创新高后，止损线上移至最高价下方15%
- 示例：买入$100 → 涨到$150 → 止损上移至$127.5

### 5.2 止盈规则

**目标价达到时**：
- 达到目标价1（+30%）：卖出1/3仓位
- 达到目标价2（+60%）：再卖出1/3仓位
- 达到目标价3（+100%+）：视基本面决定是否继续持有

**估值过高时**：
- PEG > 3x 且增速开始放缓：启动减仓
- 行业整体估值泡沫化：轮动至低估值板块

**基本面恶化时（无条件减仓）**：
- 连续两个季度营收增速显著低于预期（超过10%）
- 核心业务护城河被颠覆
- 管理层频繁大额减持
- 重大财务造假迹象

### 5.3 特殊情形处理

**黑天鹅事件**（突发政策、地缘、危机）：
- 第一时间评估：是否影响企业基本面？
- 若不影响基本面：视为买入机会，不慌乱卖出
- 若影响基本面：立即减仓，等待情况明朗

## 七、组合再平衡规则

### 7.1 定期再平衡

**时机**：
- **季度再平衡**：每季度检查行业配置比例
- **年度大调仓**：每年初根据宏观判断调整总配置

**触发条件**（任一满足即触发）：
- 单一持仓占比超过目标权重±5%
- 单一行业占比超过目标权重±10%
- 市场大幅波动后（涨跌超过20%）

### 7.2 税务效率考量

- 优先卖出亏损仓位（税收损失收割 Tax-Loss Harvesting）
- 盈利仓位尽量持有超过1年（长期资本利得税率更低）
- 避免频繁交易（交易成本侵蚀收益）
