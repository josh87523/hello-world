# AI基础设施和算力层深度分析报告

## 一、宏观背景：AI算力需求爆发与数据中心资本开支趋势

### 1.1 超级周期已至：超大规模数据中心资本开支

2026年，全球AI基础设施投资进入史无前例的超级周期。四大超大规模云计算厂商（Hyperscalers）的资本开支总额预计接近**7000亿美元**，较2025年的约3810亿美元增长约67%。这一数字令市场震惊，但所有超大规模厂商均表示其市场处于**供给受限**而非需求受限状态。

#### 四大超大规模厂商2025-2026年资本开支对比（单位：十亿美元）

| 公司 | 2025年CapEx | 2026年CapEx（指引/预估） | 同比增幅 | 主要投向 |
|------|-----------|------------------------|---------|---------|
| **Amazon** | ~$125B | ~$200B | +60% | AWS AI基础设施、数据中心 |
| **Alphabet/Google** | ~$91-93B | $175-185B | +94% | Gemini模型、TPU、Google Cloud |
| **Microsoft** | ~$80B | ~$120-145B | +50-81% | Azure AI、OpenAI合作 |
| **Meta** | ~$72B | $115-135B | +60-88% | Llama 5训练、1GW+数据中心 |
| **合计** | ~$368-370B | ~$610-665B | +65-80% | — |

**关键信号：**
- Microsoft披露Azure订单积压高达**$800亿美元**，受电力限制无法完全满足；
- 约75%的CapEx（约$4500亿）直接用于AI基础设施（GPU服务器、网络设备、数据中心建设），而非传统云计算；
- Jensen Huang声称全球数据中心年度CapEx将达到**$6000亿**以上。

### 1.2 投资逻辑：算力是AI时代的"石油"

AI大模型训练和推理对算力需求呈指数级增长。从GPT-4到GPT-5/Gemini 3，训练算力需求每18个月增长约10倍。推理端需求在2026年预计占AI芯片需求的**三分之二**，且随着AI Agent普及将进一步加速。

**核心投资框架：**
1. **GPU/加速器**：NVIDIA（垄断者）> AMD（挑战者）> 定制ASIC
2. **网络芯片**：Broadcom（交换机ASIC）> Marvell（光模块、定制芯片）
3. **代工制造**：TSMC（绝对垄断）
4. **设备供应**：ASML（EUV光刻机唯一供应商）
5. **架构/IP**：ARM Holdings（CPU架构许可）

### 2.1 基本面概览

### 2.2 Blackwell架构：AI算力的新标杆

NVIDIA的Blackwell GPU平台在FY2026取得"爆炸性"销售成绩。CEO黄仁勋表示"Blackwell销售超出预期，云端GPU已售罄"。

**Blackwell关键优势：**
- Grace Blackwell NVLink系统在推理场景下实现**比前代低一个数量级的每Token成本**；
- 72-GPU机柜级系统成为主要交付形态；
- 已在台积电亚利桑那州新晶圆厂量产，并在富士康墨西哥新工厂组装机柜级系统。

### 2.3 下一代平台：Vera Rubin

NVIDIA已向客户发送了下一代**Vera Rubin**机柜级系统的首批样品，预计2026年下半年量产。这将进一步巩固NVIDIA在训练和推理领域的领导地位。

### 2.4 战略合作深化

- **OpenAI合作**：部署至少10GW的NVIDIA系统用于下一代AI基础设施；
- **Anthropic首次采用NVIDIA**：初始1GW计算容量，使用Grace Blackwell和Vera Rubin系统；
- **Meta多年合作**：大规模部署百万级Blackwell和Rubin GPU。

### 2.5 盈利预测与估值分析

| 指标 | FY2026（实际） | FY2027E | FY2028E |
|------|--------------|---------|---------|
| 营收（十亿美元） | $215.9 | ~$310-320 | ~$400+ |
| 数据中心收入 | $193.7 | ~$311.0 | ~$400+ |
| Non-GAAP EPS | — | ~$6.5-7.0 | ~$8.5+ |
| PE（基于当前股价） | ~47.5x | ~28-30x | ~22-25x |

**Q1 FY2027指引**：收入**$780亿**（+/-2%），远超分析师预期的$726亿。

### 2.6 风险因素

- 估值虽有所回落，但绝对水平仍不低；
- 定制ASIC（Broadcom为Google/Anthropic/OpenAI设计）的份额蚕食；
- 中国市场出口限制导致收入损失（Q1 FY2027指引已不包含中国数据中心收入）；
- Vera Rubin量产节奏是否能顺利衔接Blackwell。

### 2.7 投资评级：**强烈推荐（Overweight）**

NVIDIA仍是AI算力投资的"核心中的核心"。80%的GPU市场份额、持续的架构领先、以及超大规模客户的深度绑定，使其在未来2-3年内仍无可替代。FY2027E约28-30x PE对应65%+的盈利增速，PEG不到0.5x，估值合理偏低。

## 三、AMD (AMD) 分析

### 3.1 基本面概览

| 指标 | 数据 |
|------|------|
| 当前市值 | **~$2600亿** |
| TTM市盈率 | **~63.6x** |
| 2025年营收 | ~$340亿（预计） |
| 2025年数据中心收入 | ~$160亿 |
| AI加速器年收入 | ~$95亿（2025年） |
| AI收入增长目标 | **80%+ CAGR** |
| 数据中心增长目标 | **60%+ CAGR** |

### 3.2 MI300系列与竞争力分析

MI300X在多项行业基准测试中表现**优于NVIDIA H100**，在AI推理负载下性能领先10-20%。然而，NVIDIA的Blackwell平台已大幅超越H100代际，AMD面临"追赶者困境"。

**关键进展（2026年）：**
- **OpenAI选择AMD**：部署6GW下一代AI计算容量，2026年下半年起交付**MI450 GPU**；
- **Oracle部署**：2026年起在OCI部署50,000颗AMD AI芯片；
- **MI350系列**已在neocloud客户（Crusoe、DigitalOcean、TensorWave、Vultr）中获得牵引力；
- **MI450+Helios机柜级系统**将在2026年下半年推出。

### 3.3 生态短板与风险

- ROCm软件生态仍落后于NVIDIA的CUDA，限制了企业客户大规模采用；
- NVIDIA更快的开发周期（Hopper→Blackwell→Vera Rubin）意味着AMD可能持续处于追赶状态；
- 分析师共识目标价$263，34位分析师评级为"买入"。

### 3.4 投资评级：**推荐（Buy）**，但需耐心

AMD是AI芯片市场唯一可信的GPU替代方案。OpenAI和Oracle的背书具有里程碑意义。但PE高达64x，需要MI450成功放量来支撑估值。适合在回调时建仓。

## 四、Broadcom (AVGO) 分析

### 4.1 基本面概览

| 指标 | 数据 |
|------|------|
| 当前市值 | **~$1.61万亿** |
| TTM市盈率 | **~61-75x** |
| FY2025营收 | **$639亿**（同比+24%） |
| FY2025 AI收入 | **$200亿**（同比+65%） |
| FY2026E营收 | **~$960亿**（分析师一致预期，+50%） |
| AI订单积压 | **$730亿** |

### 4.2 定制AI芯片(ASIC/XPU)：最大增长引擎

Broadcom已成为定制AI芯片领域的**绝对领导者**。随着Google、Meta等超大规模厂商寻求降低对NVIDIA的依赖并优化总拥有成本（TCO），定制ASIC的需求正在爆发。

**核心客户与订单：**
- **Google TPU**：HSBC预测2026年出货179万颗TPU，贡献约$221亿收入。Morgan Stanley预计2027年出货量达500万颗，2028年达700万颗；
- **Anthropic新订单**：2026年下半年交付价值$110亿的AI芯片；
- **OpenAI合作**：部署10GW OpenAI自研AI加速器，Citi估计该交易可带来$1000亿收入，Mizuho估计$1500-2000亿（多年期）；
- **第五个XPU客户**：$10亿订单，扩大定制芯片版图；
- **FY2026 Q1**：AI半导体收入指引$82亿，同比翻倍。

### 4.3 网络芯片：AI后端网络的关键

Broadcom的**Tomahawk 6交换机ASIC**是全球首款102.4 Tbps芯片，支持1.6T以太网端口。随着Ultra Ethernet Consortium的崛起，Broadcom是最大受益者。

### 4.4 投资评级：**强烈推荐（Overweight）** —— 2026年半导体板块首选之一

Broadcom的定制ASIC业务正处于"爆发式增长"阶段，$730亿的AI订单积压提供了极强的收入能见度。FY2026E营收增长50%，若PE在35-40x FY2027E合理估值，仍有显著上行空间。这是AI基础设施中除NVIDIA外最具确定性的投资标的。

## 五、TSMC (TSM) 分析

### 5.1 基本面概览

| 指标 | 数据 |
|------|------|
| 当前市值 | **~$1.92万亿** |
| TTM市盈率 | **~34-37x** |
| Forward市盈率 | **~24.1x** |
| 2025年营收 | 同比+33%（Q4单季$337亿） |
| 全球代工市场份额 | **72%** |
| 先进制程市场份额 | **>90%** |
| 2026年CapEx | **$520-560亿** |

### 5.2 先进制程：2nm全面量产

- **2nm产能已被完全预订**（2026年全年），客户包括Apple、Qualcomm、MediaTek、AMD、NVIDIA等；
- 台积电正在台湾兴建三座新晶圆厂以满足2nm需求，投资约$286亿；
- 3nm产能预计2026年达到极限，2nm工艺相比前代性能提升15%、功耗降低30%；
- 台积电已成功对2nm晶圆实施涨价。

### 5.3 先进封装（CoWoS）：AI芯片的瓶颈与机遇

CoWoS先进封装产能预计2026年底达到**10万-12.5万片/月**，同比增长66%（Bernstein预测），主要由NVIDIA GPU和定制ASIC需求驱动。这是AI芯片产能的关键瓶颈之一。

### 5.4 AI驱动的长期增长

- CEO预计长期收入CAGR达**25%**；
- AI加速器已贡献高两位数收入占比；
- 四大超大规模厂商2026年$6500亿+的CapEx将通过NVIDIA/AMD/Broadcom等间接流向台积电；
- 为OpenAI代工自研AI芯片（与Broadcom合作设计）。

### 5.5 投资评级：**强烈推荐（Overweight）** —— AI军备竞赛中的"军火商"

台积电是AI基础设施投资中**风险调整后回报最优**的标的。>90%的先进制程垄断意味着无论NVIDIA还是Broadcom ASIC胜出，台积电都是赢家。Forward PE仅24x，对应25%+的盈利增速，PEG不到1x，在整个AI半导体板块中估值最具吸引力。

## 六、ASML 分析

### 6.1 基本面概览

| 指标 | 数据 |
|------|------|
| 当前市值 | **~$4760亿** |
| TTM市盈率 | **~52-55x** |
| Forward市盈率 | **~43x** |
| 2025年营收 | **EUR 327亿**（创纪录，毛利率52.8%） |
| 2026E营收指引 | **EUR 340-390亿** |
| 2030年营收目标 | EUR 440-600亿 |

### 6.2 EUV光刻机垄断地位

ASML是全球**唯一**的高端极紫外光（EUV）光刻机制造商。所有先进晶圆厂（台积电、三星、Intel）都依赖其EUV系统生产最先进芯片。

**技术突破：** ASML发布的最新EUV技术可将芯片产出提高**50%**，进一步拉大与潜在竞争对手的差距。High NA EUV系统已进入客户验证阶段。

### 6.3 增长驱动

- EUV工具占积压订单的**65%**，较上年的62%提升；
- 每代工艺节点需要更多EUV光刻层，驱动单位需求增长；
- DRAM厂商正在增加EUV光刻层用于HBM生产；
- AI相关需求的可持续性得到客户更乐观的评估。

### 6.4 风险：中国销售下滑

ASML预计2026年中国市场销售将较2024-2025年的"非常强劲"水平**显著下降**，这是主要的收入逆风。此外，地缘政治不确定性和出口限制也是持续风险。

### 6.5 投资评级：**推荐（Buy）**

ASML的垄断地位无可动摇，长期增长逻辑清晰。但当前PE约52-55x偏高，且中国收入下降是短期压力。适合长期持有，可在回调时加仓。

## 九、AI芯片竞争格局与供应链分析

### 9.1 竞争格局总览

```
GPU通用加速器                    定制ASIC
┌─────────────────┐           ┌─────────────────┐
│ NVIDIA (~80%)    │           │ Broadcom (XPU)   │
│ - Blackwell/Rubin│           │ - Google TPU     │
│ AMD (~10-15%)    │           │ - Anthropic      │
│ - MI300/MI450    │           │ - OpenAI自研     │
│ Intel Gaudi (~2%)│           │ Marvell（定制芯片）│
└─────────────────┘           └─────────────────┘
         │                              │
         ▼                              ▼
    ┌──────────────────────────────────────┐
    │        TSMC（>90%先进制程代工）         │
    │        ASML（100% EUV光刻机）          │
    └──────────────────────────────────────┘
```

### 9.2 供应链关键瓶颈

1. **CoWoS先进封装**：台积电产能是核心瓶颈，预计2026年底月产能达10-12.5万片；
2. **HBM（高带宽内存）**：SK海力士和三星产能限制GPU整体交付；
3. **电力供应**：数据中心电力需求激增，成为扩张的物理限制；
4. **EUV光刻机交付**：ASML的产能决定了先进制程的扩张速度。

### 9.3 GPU vs ASIC的竞争动态

| 维度 | GPU（NVIDIA） | 定制ASIC（Broadcom等） |
|------|-------------|---------------------|
| 通用性 | 高（多种工作负载） | 低（针对特定模型优化） |
| 性能/功耗比 | 良好 | 更优（特定任务） |
| TCO | 较高 | 较低（大规模部署时） |
| 上市时间 | 快（标准产品） | 慢（定制设计18-24个月） |
| 生态系统 | CUDA生态完善 | 需自建软件栈 |
| 适用客户 | 所有规模 | 仅超大规模厂商 |

**趋势判断：** GPU和ASIC将长期共存。NVIDIA在训练和通用推理领域保持主导，ASIC在超大规模厂商的特定推理任务中份额将持续提升。Broadcom预计2026年下半年推理需求将加速增长。
