## 一、全球互联网广告市场趋势

### 1.1 市场规模与增长

2025年全球数字广告市场规模预计达到约**$7400亿美元**，2026年预计增长至约**$8200亿美元**（+11%）。AI技术正在从三个维度重塑广告市场：

1. **精准投放**：AI驱动的广告投放系统（如Meta的Andromeda、Google的Performance Max）大幅提升广告ROI；
2. **创意自动化**：AI生成广告素材降低广告主创建成本，扩大中小广告主参与度；
3. **新广告形态**：AI搜索、AI Agent交互中嵌入的新型广告位正在出现。

### 1.2 市场份额格局

| 平台 | 全球份额（2025估计） | 趋势 |
|------|---------------------|------|
| Google | ~28-30% | 稳定，AI搜索变革中 |
| Meta | ~22-24% | 上升，AI广告优化强劲 |
| Amazon | ~7-8% | 快速上升 |
| TikTok/字节 | ~5-6% | 上升，但面临地缘风险 |
| Apple | ~2-3% | 稳步上升（ASA） |
| 其他 | ~30-35% | 分散 |

## 二、Apple (AAPL) —— AI手机与服务生态

### 2.1 基本面概览

| 指标 | 数据 |
|------|------|
| 当前市值 | **~$3.8-4.0万亿**（全球最高之一） |
| FQ1 2026营收（截至2025.12） | **$1438亿**（同比+16%） |
| FQ1 2026 EPS | **$2.84**（同比+19%） |
| 服务收入 | **$300亿**（历史新高） |
| 毛利率指引 | 48-49% |
| 分析师一致评级 | 强烈买入，中位目标价$300 |

### 2.2 AI手机：iPhone超级周期

- **iPhone 17**是2026年最重要的消费电子产品发布之一。分析师预测iPhone收入同比增长12%+，达$780-800亿/季度水平；
- 约**3亿+用户**持有4年以上旧款iPhone，构成庞大的换机潜力池；
- Apple Intelligence（增强版Siri和AI功能）将只在搭载最新处理器的设备上完整运行，推动硬件升级；
- Morgan Stanley预计2026年将是Apple正式进入扩展AI功能的年份，可能与Google Gemini合作。

### 2.3 服务收入：最亮眼的增长点

服务收入持续保持两位数增长，包括App Store、Apple Music、Apple TV+、iCloud、Apple Pay以及Google搜索默认引擎的付费。服务业务是Apple利润率最高的部分。

### 2.4 估值与风险

- PE约32-35x，在大型科技股中偏高（相对于收入增速而言）；
- AI投入相对保守（年度CapEx约$140亿，远低于其他超大规模厂商）；
- Apple Car项目已于2024年取消，Vision Pro VR头显市场反响平淡；
- 2026年底或2027年初可能推出的**智能眼镜**是下一个潜在爆品。

### 2.5 投资评级：**推荐（Buy）**，但估值需注意

Apple的服务生态和iPhone换机周期提供了稳健的增长基础。但PE偏高且AI投入相对不足，在AI军备竞赛中可能逐渐落后。建议作为稳健配置而非进攻性仓位。

## 三、Netflix (NFLX) —— 广告与内容的新增长极

### 3.1 基本面概览

| 指标 | 数据 |
|------|------|
| 全球付费用户 | **~3.25亿** |
| 2026年营收指引 | **$507-517亿** |
| 2026年运营利润率目标 | **31.5%** |
| 2025年广告收入 | **>$15亿**（同比翻倍） |
| 2026年广告收入预期 | **~$30亿**（再翻倍） |
| Q4 2025营收 | **$120.5亿**（同比+17.6%） |
| Q4 2025 EPS | **$0.56**（同比+30.2%） |
| 2026E EPS | **$3.13**（+24%） |

### 3.2 广告业务：早期但高成长

- 在有广告的市场中，广告套餐占新注册用户的**50%**；
- 广告用户季度环比增长**35%**；
- Netflix目前仅占可寻址市场的**7%**，增长空间巨大；
- Wedbush预计广告收入2026年至少翻倍至**$30亿**，2027年继续加速。

### 3.3 内容与新业务

- 2026年内容摊销预计增长约10%，但增速低于收入增速，支撑利润率扩张；
- 云游戏和移动端新界面是2026年的重要产品节点；
- 可能收购Warner Bros.（消息面）将是重大催化剂。

### 3.4 投资评级：**推荐（Buy）**

Netflix是流媒体领域的绝对领导者。广告业务从零到$30亿的增长轨迹、31.5%运营利润率目标、以及仅7%的市场渗透率，提供了清晰的中期增长路径。

| 8 | 快手 | ~$25B | +25% |

### 5.2 港股 vs 美股估值折价

恒生科技指数平均预测PE约**18倍**，远低于纳斯达克100指数的**26.87倍**。具体折价情况：

| 对比维度 | 港股互联网均值 | 美股互联网均值 | 折价幅度 |
|---------|-------------|-------------|---------|
| Forward PE | 15-20x | 25-35x | **30-45%** |
| PS（市销率） | 3-5x | 6-12x | **40-60%** |
| EV/EBITDA | 8-12x | 15-25x | **40-55%** |

**折价原因：**
1. 中国经济增速放缓和消费信心不足；
2. 监管政策不确定性（虽已大幅缓解）；
3. 地缘政治风险溢价（中美关系）；
4. 部分公司（美团、京东）受外卖大战拖累利润率；
5. 港股市场流动性不足。

**折价收窄的催化剂：**
1. AI商业化进展（中国AI大模型快速追赶）；
2. "十五五"政策支持科技自主；
3. 美联储降息周期改善港股流动性；
4. 中概股回流增加港股资产稀缺性；
5. 企业大规模回购持续。

### 5.3 港股互联网回购潮分析

2025年港股回购规模创历史新高，但2026年初出现**降温迹象**：

- 2026年初至2月23日，港股回购总额约**183.74亿港元**，仅为2025年同期**320.96亿港元的57%**；
- 回购降温主要原因：估值回升使"低位扫货"动力减弱，叠加AI研发投入占用资金；
- **跨市场分流**：阿里巴巴在美股回购$10.46亿（约81.79亿港元），中概股偏好在美股而非港股回购；
- AH股公司（如宁德时代）倾向在A股回购以铺路再融资。

**结论：** 虽然回购量有所下降，但头部企业的常态化回购计划（如阿里$191亿美元剩余额度、腾讯持续回购）仍为港股市值构筑坚实底线。

## 七、阿里巴巴 (9988.HK) 分析

### 7.1 基本面概览

| 指标 | 数据 |
|------|------|
| 当前股价 | ~145-150港元（估计） |
| TTM市盈率 | **~16-21x** |
| 2025 Q3（9月季度）云智能收入 | 增速加速至**34.5%** |
| AI相关云收入占比 | **>20%**（保持三位数增长） |
| 通义千问模型下载量 | **>6亿** |
| 3年CapEx计划 | **3800亿元+**（AI基础设施） |
| 分析师目标价均值 | **188港元**（+27%上行） |
| 未动用回购额度 | **$191亿美元**（至2027年3月） |

### 7.2 核心看点

#### A. 云智能：AI变现加速

- 过去3个季度云收入增速从13%加速至18%再到**26-34.5%**，AI是核心驱动力；
- 中国超50%的大模型运行在阿里云，全球第四/亚洲第一（Gartner）；
- 通义千问（Qwen）衍生模型超17万个；
- 平头哥自研芯片（含义AI推理芯片）潜在IPO可释放价值。

#### B. 电商：稳中有升

- 中国电商集团收入**1400.72亿元**（同比+10%）；
- "全站推广"产品和0.6%技术服务费将逐步提升货币化率；
- 即时零售（饿了么）短期亏损（FY2026预计超700亿），但中长期GMV空间超1万亿。

#### C. 资本返还

- FY2025全年回购$119亿、派息$46亿，彰显管理层信心；
- 剩余回购额度$191亿，有效期至2027年3月。

### 7.3 估值分析

- 当前PE约16-21x，远低于中国电商行业平均38.98x；
- 国投证券采用SOTP估值法，目标价**231港元/238美元**；
- 招商证券国际认为阿里是港股科技板块估值修复的**首选标的**。

### 7.4 投资评级：**强烈推荐（Overweight）** —— 估值修复空间最大

阿里巴巴是港股中**估值修复空间最大**的互联网标的。云智能加速增长（34.5%）、AI商业化领先、大规模回购支撑，PE仅16-21x处于历史低位。主要风险是外卖大战带来的短期利润压力和宏观经济不确定性。

美团当前股价已接近52周低点，分析师一致目标价隐含44%上行空间。2026年若外卖补贴边际收缩，利润修复将成为核心催化剂。但短期利润不确定性较高，建议小仓位布局，等待利润拐点信号。

## 十六、港股配置策略总结

### 16.1 核心观点

1. **港股互联网当前是全球最具性价比的科技投资区域之一**。恒生科技PE 18x vs 纳斯达克100的27x，折价33%。

2. **2026年港股有望迎来"戴维斯双击"**：估值修复（PE扩张） + 盈利增长（AI商业化+外卖大战缓和）。

3. **配置节奏建议**：遵循"算力筑基 --> 应用开花 --> 终端兑现"的产业节奏：
   - Q1-Q2：布局AI基础设施受益标的（阿里云、腾讯AI）
   - Q3-Q4：关注应用落地变现（视频号商业化、可灵AI、新游上线）

4. **美股互联网中Uber被严重低估**（PE仅11x），是平台经济中性价比最高的标的。

### 16.2 风险提示

1. **地缘政治风险**：中美关系恶化可能导致中概股进一步估值压缩；
2. **宏观经济风险**：中国消费信心不足、经济增速放缓；
3. **外卖大战持续**：若补贴战延续至2026年下半年，美团/京东利润持续承压；
4. **监管风险**：虽已大幅缓解，但政策不确定性仍存在；
5. **港股流动性风险**：成交量不足可能放大波动性；
6. **汇率风险**：人民币贬值对以港币计价的资产构成压力。
