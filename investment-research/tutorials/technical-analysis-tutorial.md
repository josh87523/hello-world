## 前言：为什么要学技术分析？

技术分析不是算命，也不是万能公式，而是一套**用历史价格和成交量数据判断市场供需关系**的方法论。

**基本面分析告诉你"买什么"**，**技术分析告诉你"何时买"**。

两者结合，才能实现：
- 用基本面筛选出优质标的
- 用技术面找到合适的买卖时机
- 用风险管理保护本金

## 第一章：技术分析核心理论

### 1.1 道氏理论（Dow Theory）—— 一切的起点

查尔斯·道（Charles Dow）于19世纪末提出，是现代技术分析的理论基础。

**六大核心假设**：

1. **市场包含一切信息**：价格已经反映了所有已知信息（类似有效市场假说的弱式版本）
2. **市场存在三种趋势**：
   - **主趋势**（Primary Trend）：持续1年以上的大趋势
   - **次级趋势**（Secondary Trend）：持续数周至数月的中期反转
   - **短期波动**（Minor Trends）：日内至数天的短期波动
3. **主趋势分三个阶段**：
   - 积累期（聪明钱悄悄建仓）
   - 公众参与期（趋势确立，大众跟进）
   - 分配期（聪明钱悄悄出货）
4. **指数必须相互确认**：工业指数和运输指数必须同时发出信号才有效
5. **成交量必须确认趋势**：趋势方向成交量应放大
6. **趋势持续直到明确反转**：不要轻易预测趋势终结

**实战意义**：
- 顺势交易，不要试图预测顶底
- 等待趋势确认后再入场，不要抢跑
- 用成交量验证价格走势的真实性

### 1.2 三大核心假设

技术分析的一切建立在三个假设上：

```
假设1: 市场行为包含一切 (Market action discounts everything)
  → 所有基本面、政策、情绪已反映在价格中
  → 因此，分析价格本身就足够了

假设2: 价格以趋势方式运动 (Prices move in trends)
  → 趋势一旦形成，延续概率大于反转概率
  → "趋势是你的朋友"（The trend is your friend）

假设3: 历史会重演 (History repeats itself)
  → 人类情绪（贪婪/恐惧）驱动市场行为
  → 相似的情绪在图表上形成相似的形态
```

## 第二章：K线图基础

### 2.1 K线（蜡烛图）解读

K线是最基础的价格可视化工具，源自日本大米市场（17世纪）。

```
        │  ← 上影线（当日最高价 vs 开盘/收盘最高者）
       ┌┴┐
       │ │  ← 实体（开盘价到收盘价的区间）
       │ │    阳线（绿/白）：收盘 > 开盘
       │ │    阴线（红/黑）：收盘 < 开盘
       └┬┘
        │  ← 下影线（当日最低价 vs 开盘/收盘最低者）
```

**K线四个关键价格**：
- **开盘价（Open）**：当天第一笔交易价格
- **最高价（High）**：当天最高成交价
- **最低价（Low）**：当天最低成交价
- **收盘价（Close）**：当天最后一笔成交价

### 2.2 重要K线形态

#### 单根K线形态

| 形态 | 外观 | 信号 | 可靠性 |
|------|------|------|-------|
| **大阳线** | 长实体，无影线 | 强势上涨信号 | ⭐⭐⭐ |
| **大阴线** | 长实体，无影线 | 强势下跌信号 | ⭐⭐⭐ |
| **十字星（Doji）** | 几乎无实体 | 多空争夺，趋势可能反转 | ⭐⭐⭐ |
| **锤子线（Hammer）** | 小实体，长下影线 | 底部反转信号（阳线更强） | ⭐⭐⭐⭐ |
| **射击之星（Shooting Star）** | 小实体，长上影线 | 顶部反转信号 | ⭐⭐⭐⭐ |
| **纺锤线（Spinning Top）** | 小实体，上下影线 | 市场犹豫不决 | ⭐⭐ |

**锤子线详解**（重要的底部信号）：
```
  │  ← 很短的上影线（或无）
 ┌┴┐
 │ │  ← 小实体（阳线更可靠）
 └┬┘
  │
  │
  │  ← 长下影线（至少是实体2倍）
  │
```
- 出现在下跌趋势末端
- 表明卖盘已经减弱，买盘在低位承接
- 确认：次日出现阳线

#### 两根K线组合形态

| 形态 | 信号 | 出现位置 | 说明 |
|------|------|---------|------|
| **吞没形态（Bullish Engulfing）** | 底部买入 | 下跌末端 | 阳线实体完全吞没前一根阴线 |
| **看跌吞没（Bearish Engulfing）** | 顶部卖出 | 上涨末端 | 阴线实体完全吞没前一根阳线 |
| **早晨之星（Morning Star）** | 底部买入 | 下跌末端 | 三根K线：大阴→小实体→大阳 |
| **黄昏之星（Evening Star）** | 顶部卖出 | 上涨末端 | 三根K线：大阳→小实体→大阴 |
| **看涨平头（Bullish Harami）** | 可能反转 | 下跌末端 | 小阳线被大阴线包含 |

## 第三章：趋势分析

### 3.1 趋势的定义与识别

**上升趋势**：一系列更高的高点（Higher Highs）和更高的低点（Higher Lows）

```
价格
  ↑     高点3
  |   高点2  ×
  | 高点1  ×   低点3
  |   ×   低点2  ×
  | 低点1  ×
  |   ×
  +─────────────→ 时间
```

**下降趋势**：一系列更低的高点（Lower Highs）和更低的低点（Lower Lows）

**横盘震荡**：高点和低点在相近水平波动

### 3.2 趋势线的绘制

**上升趋势线**：
- 连接两个或更多**低点**（回调低点）
- 线越长、触碰点越多，越有效
- 价格跌破趋势线 → 可能趋势改变

**下降趋势线**：
- 连接两个或更多**高点**（反弹高点）
- 价格突破趋势线（向上）→ 可能趋势改变

**趋势线的意义**：
- 斜率越陡，趋势越不可持续（迟早修正）
- 触碰次数越多，突破时信号越强
- 配合成交量：突破时成交量放大更可信

### 3.3 支撑与阻力

**支撑位（Support）**：价格下跌到某一水平后反弹的价格区域
- 买盘强于卖盘的价格区域
- 历史低点、整数关口、均线等

**阻力位（Resistance）**：价格上涨到某一水平后回落的价格区域
- 卖盘强于买盘的价格区域
- 历史高点、整数关口、均线等

**重要规则：支撑与阻力可以互换**
- 被有效突破的阻力位 → 变成新的支撑位
- 被有效跌破的支撑位 → 变成新的阻力位

```
价格
  ↑         |←阻力→|
  |   ×  ×  阻力线  突破后
  |  ×    × ─────── ←变为支撑
  |      支撑线
  |   ×  ×  ─────── ←支撑
  +─────────────────→ 时间
```

## 第四章：移动平均线

### 4.1 移动平均线基础

移动平均线（MA）是一定周期内价格的平均值，用于平滑短期波动，识别趋势方向。

**常用周期**：

| 周期 | 类型 | 用途 |
|------|------|------|
| 5日/10日 | 短期 | 短线交易参考 |
| **20日** | 短中期 | 最常用，代表"一个月均价" |
| **50日** | 中期 | 机构常用，代表"季度均价" |
| **200日** | 长期 | 判断多空分界线，牛熊分野 |

**计算方法**：

- **简单移动平均（SMA）**：各期收盘价的算术平均
- **指数移动平均（EMA）**：对近期数据给予更高权重，对价格变化反应更快

### 4.2 均线交叉信号

**黄金交叉（Golden Cross）**：短期均线向上穿越长期均线
- 典型：50日均线上穿200日均线
- 信号：**看多**，趋势可能转为上升

**死亡交叉（Death Cross）**：短期均线向下穿越长期均线
- 典型：50日均线下穿200日均线
- 信号：**看空**，趋势可能转为下降

**注意**：均线交叉是滞后指标，信号确认时价格已经移动了一段距离。

### 4.3 均线支撑与阻力

均线在趋势中扮演动态支撑/阻力的角色：

**上升趋势中**：
- 价格回调到20日均线附近 → 短线买入机会
- 价格回调到50日均线附近 → 中期买入机会
- 价格跌破200日均线 → 需要重新评估趋势

**下降趋势中**：
- 价格反弹到20日均线附近 → 短线卖出信号
- 价格反弹到50日均线附近 → 做空加仓时机

### 4.4 均线多头排列与空头排列

```
多头排列（价格 > 20MA > 50MA > 200MA）
→ 趋势健康，逢回调买入

价格 ↑
      价格
        20MA（短期）
          50MA（中期）
            200MA（长期）

空头排列（价格 < 20MA < 50MA < 200MA）
→ 趋势向下，反弹卖出
```

## 第五章：量价关系

### 5.1 成交量的意义

**成交量 = 市场的"燃料"**

- 量升价涨：趋势健康，继续做多
- 量缩价涨：动力不足，上涨可能乏力（需警惕）
- 量升价跌：卖压沉重，下跌趋势确认
- 量缩价跌：卖盘衰竭，可能接近底部

### 5.2 量价背离

**顶背离（看跌）**：
- 价格创新高，但成交量没有创新高
- 说明上涨动能减弱，可能见顶

**底背离（看涨）**：
- 价格创新低，但成交量没有创新低
- 说明卖盘减弱，可能见底

### 5.3 重要的成交量形态

**放量突破**：
- 价格突破关键阻力位同时成交量大幅放大
- 信号强度高，假突破概率低

**缩量回调**：
- 在上升趋势中，回调期间成交量明显减少
- 正常的健康回调，不改变趋势

**放量下跌后的止跌信号**：
- 恐慌性放量杀跌后出现大阳线
- 常见于熊市末期的"最后一跌"

## 第六章：技术指标详解

### 6.1 RSI（相对强弱指标）

**定义**：衡量价格上涨动能与下跌动能的对比，范围0-100

**计算逻辑**：
```
RSI = 100 - (100 / (1 + RS))
RS = 平均上涨幅度 / 平均下跌幅度（通常取14日）
```

**标准解读**：

| RSI值 | 市场状态 | 操作建议 |
|-------|---------|---------|
| > 80 | 严重超买 | 考虑减仓，等待回调 |
| 70-80 | 超买 | 谨慎追高，注意风险 |
| 50-70 | 中性偏多 | 正常持有，顺势操作 |
| 30-50 | 中性偏空 | 观望为主 |
| 20-30 | 超卖 | 关注买入机会 |
| < 20 | 严重超卖 | 可能接近底部，小仓试探 |

**RSI背离（重要！）**：
- **顶背离**：价格创新高，RSI未创新高 → 看跌信号
- **底背离**：价格创新低，RSI未创新低 → 看涨信号

**注意**：超买不代表立即下跌，强势上涨时RSI可以长期处于70以上。

### 6.2 MACD（指数移动平均线聚散）

**组成**：
- **MACD线**（快线）：12日EMA - 26日EMA
- **信号线**（慢线）：MACD线的9日EMA
- **柱状图（Histogram）**：MACD线 - 信号线

**关键信号**：

1. **零线以上/以下**：
   - MACD线在零线以上 → 整体偏多
   - MACD线在零线以下 → 整体偏空

2. **金叉/死叉**：
   - MACD线上穿信号线（金叉）→ 买入信号
   - MACD线下穿信号线（死叉）→ 卖出信号

3. **MACD背离**（比RSI背离更可靠）：
   - 顶背离：价格新高，MACD未新高
   - 底背离：价格新低，MACD未新低

4. **柱状图收缩**：
   - 柱状图持续缩短 → 趋势动能减弱
   - 柱状图由负转正 → 多头开始占优

### 6.3 布林带（Bollinger Bands）

**组成**：
- **中轨**：20日简单移动平均线（SMA20）
- **上轨**：中轨 + 2倍标准差
- **下轨**：中轨 - 2倍标准差

**解读**：

| 价格位置 | 含义 | 操作参考 |
|---------|------|---------|
| 触碰上轨 | 价格处于相对高位 | 注意回调风险 |
| 在中轨以上 | 短期偏多 | 持有/追多 |
| 触碰中轨 | 回调到均线支撑 | 可能的买入点 |
| 在中轨以下 | 短期偏空 | 观望/减仓 |
| 触碰下轨 | 价格处于相对低位 | 关注反弹机会 |

**布林带宽度（Bandwidth）**：
- **布林带收窄（Squeeze）**：波动率降低，大行情即将来临
- **布林带扩张**：波动率增加，趋势确立

**突破信号**：
- 布林带收窄后，价格突破上轨 → 做多
- 布林带收窄后，价格跌破下轨 → 做空

### 6.4 随机指标（KD / Stochastic）

**定义**：比较当前收盘价相对于一定周期内价格区间的位置

**组成**：
- **%K线**（快线）：当前收盘价在N日价格区间的位置
- **%D线**（慢线）：K线的3日移动平均

**解读**：

| KD值 | 状态 | 信号 |
|------|------|------|
| > 80 | 超买区 | 关注卖出时机 |
| 20-80 | 中性区 | 无明确信号 |
| < 20 | 超卖区 | 关注买入时机 |

**金叉/死叉**：
- K线上穿D线（超卖区）→ 买入信号
- K线下穿D线（超买区）→ 卖出信号

## 第七章：经典图表形态

### 7.1 反转形态

#### 头肩顶（Head and Shoulders Top）
**信号**：看跌反转，趋势由涨转跌

```
              头部（最高点）
             /↑\
    左肩 /↑\   /↑\ 右肩
        /   \ /   \
颈线  ───────────────────
```

**操作**：
- 识别：左肩→头部→右肩（右肩不创新高）
- 入场：价格有效跌破颈线
- 止损：右肩高点以上
- 目标：颈线以下，高度等于头部到颈线距离

#### 双底（Double Bottom / W底）
**信号**：看涨反转，趋势由跌转涨

```
      颈线（阻力位）
  ─────────────────────
    \       /
     \  /\ /
      \/  \/
     底1  底2（相近价格）
```

**操作**：
- 识别：两个相近低点，之间有一次反弹
- 入场：价格有效突破颈线（放量更佳）
- 止损：第二个底部以下
- 目标：颈线以上，高度等于底到颈线距离

#### 双顶（Double Top / M顶）
**信号**：看跌反转，趋势由涨转跌

```
     顶1  顶2（相近价格）
      /\  /\
     /  \/  \
  ─────────────────────
      颈线（支撑位）
```

### 7.2 持续形态（中继形态）

#### 上升三角形（Ascending Triangle）
```
阻力线 ─────────────────
       /\/\/\/\
      支撑线（越来越高的低点）
→ 信号：看涨，突破阻力线入场
```

#### 下降三角形（Descending Triangle）
```
       高点越来越低
        /\/\/\
支撑线 ─────────────────
→ 信号：看跌，跌破支撑线做空
```

#### 旗形整理（Flag / Pennant）
```
上升旗形（Bull Flag）：
  大涨 ↑↑↑
       ↘ 缓慢回调（旗形）
       ↗ 突破继续上涨

→ 信号：看涨，突破旗形上边界
```

**旗形特征**：
- 前期有急剧的"旗杆"走势（强烈趋势）
- 之后出现相对平静的回调（旗形）
- 回调期间成交量应明显缩小
- 突破时成交量放大，延续原有趋势

## 第八章：实战交易策略

### 8.1 趋势跟踪策略

**核心理念**：顺势而为，不预测顶底

**操作步骤**：

```
第1步：确认趋势
  - 价格在200日均线以上 → 大趋势向上
  - 均线多头排列（20MA > 50MA > 200MA）
  - ADX > 25 → 趋势有力

第2步：等待回调
  - 价格从高位回调10-15%
  - 回调到均线支撑（20MA或50MA）
  - 回调成交量应缩小

第3步：寻找入场信号
  - K线出现止跌信号（锤子线、十字星、吞没形态）
  - RSI回落到40-50区间（超卖附近）
  - MACD出现底背离或金叉

第4步：执行买入
  - 小仓位先买（1/3），等待确认
  - 价格重新站上20MA，加仓（1/3）
  - 趋势明确恢复，再次加仓（1/3）

第5步：设置止损和目标
  - 止损：回调低点以下5-8%
  - 目标：前期高点，或1:2/1:3的风险回报比
```

### 8.2 突破策略

**核心理念**：在关键阻力位突破时入场

**操作步骤**：

```
识别阻力位：
  - 历史高点、52周高点
  - 整数关口（$100、$200等）
  - 长期下降趋势线

等待突破条件：
  - 价格在阻力位下方横盘整理（越久越好）
  - 成交量在横盘期间逐渐减少
  - 出现"弹簧效应"（短暂跌破后迅速拉回）

突破确认：
  - 收盘价明确站上阻力位（不是盘中触碰）
  - 成交量比过去30日均量放大50%以上
  - 突破当日或次日不回落

入场：
  - 突破日收盘买入，或次日开盘买入
  - 或设置限价单在阻力位上方1-2%买入

止损：
  - 设在阻力位以下3-5%（假突破止损）
```

### 8.3 支撑反弹策略

**核心理念**：在强支撑位等待反弹

```
识别强支撑：
  - 历史重要低点（多次测试未跌破）
  - 长期均线（200日均线）
  - 斐波那契回撤关键位（38.2%、50%、61.8%）
  - 整数关口

等待反转信号（缺一不可）：
  - K线止跌形态（锤子线、早晨之星）
  - RSI超卖（< 30）
  - 成交量配合（止跌时放量更佳）

入场操作：
  - 确认信号后次日买入
  - 分批建仓，第一批30%

止损设置：
  - 跌破支撑位5%以下止损
```

### 8.4 斐波那契回撤应用

斐波那契数列在金融市场中广泛应用，关键回撤位：

| 回撤比例 | 重要性 | 说明 |
|---------|-------|------|
| 23.6% | 低 | 强势上涨的轻微回调 |
| **38.2%** | 高 | 重要支撑/阻力位 |
| **50.0%** | 极高 | 最常被市场关注 |
| **61.8%** | 极高 | 黄金分割比，最强支撑 |
| 78.6% | 中 | 深度回调支撑 |

**如何使用**：
1. 确认一段明显的上涨（或下跌）行情
2. 在图表工具中标注起点和终点
3. 系统自动生成各斐波那契水平
4. 价格回调到这些水平附近时关注买入机会

## 第九章：多周期分析框架

### 9.1 为什么需要多周期分析？

不同时间周期的趋势可能相互矛盾：
- 日线看涨，但周线仍在下降趋势中
- 月线是牛市，但日线已经超买

**解决方法**：大周期决定方向，小周期决定时机

### 9.2 自上而下分析框架（Top-Down）

```
月线图 → 判断长期趋势和关键支撑阻力
  ↓
周线图 → 判断中期趋势，寻找建仓时机
  ↓
日线图 → 精确入场点，设置止损
  ↓
60/15分钟图 → 超短线操作（不推荐新手）
```

**操作流程**：

1. **月线**：确认处于长期上升趋势，价格在200周均线以上
2. **周线**：确认中期趋势向上，寻找支撑回调
3. **日线**：在周线支撑位附近，日线出现K线反转信号，执行买入

### 9.3 不同时间框架的使用建议

| 投资风格 | 主要参考周期 | 辅助周期 | 持仓时间 |
|---------|-----------|---------|---------|
| 长期投资 | 月线/周线 | 日线入场 | 1年以上 |
| 中期波段 | 周线/日线 | 4小时入场 | 1-12周 |
| 短线交易 | 日线 | 1小时入场 | 1-20天 |
| 超短线 | 60分钟 | 15分钟 | 日内 |

## 第十章：技术分析的局限性与注意事项

### 10.1 技术分析不能做什么

1. **无法预测黑天鹅事件**：政策突变、财务造假、突发危机不会在K线上提前显示
2. **自我实现的预言**：大量交易者关注同一形态，会增加形态的"实现概率"
3. **无法准确预测时间**：可以判断方向，但很难预测"何时"到达目标
4. **散户vs机构的博弈**：机构了解散户的止损区域，可能故意触发止损

### 10.2 常见误区

**误区1**：技术分析可以单独使用
- 正确做法：结合基本面，技术面只是选择时机的工具

**误区2**：越多指标越准确
- 正确做法：精通3-5个核心指标胜过粗通20个指标
- 指标越多，相互矛盾的信号越多，反而更难决策

**误区3**：在任何市场都可以用
- 流动性不足的股票，技术分析可靠性大幅降低
- 消息面主导的市场，技术信号会失效

**误区4**：历史形态100%可靠
- 任何技术信号都有失败概率
- 好的交易者接受失败，并通过止损控制损失

### 10.3 如何提高技术分析准确性

1. **与基本面结合**：技术信号出现在基本面优质标的上，可靠性更高
2. **多周期共振**：日线、周线同时出现同向信号，可靠性翻倍
3. **成交量确认**：所有价格信号都应用成交量验证
4. **只做高胜率设置**：不追求数量，只等待最确定的机会
5. **严格止损**：错误时快速离场，正确时让利润奔跑

### 信号可靠性排序

```
高可靠性（多因素共振）：
  ✅ 月线+周线+日线趋势一致
  ✅ 量价配合良好
  ✅ 多个指标同向共振
  ✅ 在关键支撑/阻力位出现信号

中可靠性：
  ⚠️ 单个周期信号
  ⚠️ 成交量不明显

低可靠性（谨慎对待）：
  ❌ 逆趋势交易
  ❌ 无成交量支撑的突破
  ❌ 单一指标信号，其他指标矛盾
```

### 交易决策检查清单

在每次入场前检查：
- [ ] 大趋势方向是什么？（周线/月线）
- [ ] 目前价格在哪个支撑/阻力位附近？
- [ ] RSI处于什么区间？（超买/超卖/中性）
- [ ] MACD是金叉还是死叉？
- [ ] 成交量与价格走势是否配合？
- [ ] 止损位在哪里？（跌破即离场）
- [ ] 风险回报比是否至少1:2？
- [ ] 这笔交易与大趋势方向一致吗？
