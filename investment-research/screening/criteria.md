## 一、量化筛选标准总览

### 1.1 核心财务指标阈值

| 指标 | 成长股标准 | 价值股标准 | 质量股标准 | 说明 |
|------|-----------|-----------|-----------|------|
| 营收增长率（YoY） | >= 15% | >= 5% | >= 8% | 最近四个季度同比增长 |
| 净利润增长率（YoY） | >= 20% | >= 10% | >= 12% | 扣非净利润优先 |

| 净利率 | >= 10% | >= 8% | >= 15% | 排除周期性低点 |
| ROE（净资产收益率） | >= 15% | >= 12% | >= 20% | 连续3年达标为佳 |
| ROIC（投入资本回报率） | >= 12% | >= 10% | >= 15% | 更能反映真实回报 |

## 二、美股筛选标准

### 2.1 美股特殊筛选条件

**市场环境参数（2026年）：**
- S&P 500一致预期EPS增长率：~14.4%
- 华尔街主要投行S&P 500年终目标：7,100-8,100点
- 联储利率预期：2026年预计降息两次
- AI基础设施支出持续增长，半导体行业销售额预计首次突破1万亿美元

**基本面筛选：**
- 市值 >= 20亿美元（中大盘优先，小盘需额外评估）
- 日均成交额 >= 1,000万美元
- 分析师覆盖 >= 5家
- S&P评级 >= BBB-（投资级）或无评级但现金流稳健

**行业偏好（2026年重点）：**

| 优先级 | 行业 | 逻辑 |
|--------|------|------|

| 最高 | AI应用与软件 | 从资本支出转向应用变现，企业AI采纳加速 |
| 高 | 云计算与SaaS | AI驱动云消费增长，企业数字化转型 |

## 四、成长股筛选模型

### 4.1 CAN-SLIM改良模型

本模型基于William O'Neil的CAN-SLIM体系，针对2026年市场环境做出优化：

**C - 当季每股盈利（Current Quarterly EPS）**
- EPS同比增长 >= 25%
- 连续2个季度加速增长为佳
- 营收增长 >= 20%同步验证

**A - 年度盈利增长（Annual Earnings Growth）**
- 过去3年EPS复合增长率 >= 20%
- ROE >= 17%
- 营收复合增长率 >= 15%

**N - 新产品/新管理/新高（New）**
- 有AI相关新产品或服务推出
- 股价处于或接近52周新高
- 市场尚未完全反映的新催化剂

**S - 供需关系（Supply and Demand）**
- 公司实施回购计划
- 机构持股季度环比增加
- 内部人未大规模减持

**L - 领涨股特征（Leader）**
- 所在行业为市场当前领涨板块
- 相对强度排名前20%
- 公司为行业细分龙头

**I - 机构赞助（Institutional Sponsorship）**
- 顶级基金持仓（如Berkshire、Pershing Square、Duquesne等）
- 分析师一致预期"强力买入"
- 近期获得知名对冲基金新建仓

**M - 市场方向（Market Direction）**
- S&P 500处于上升趋势
- 恒生指数处于上升趋势
- 市场广度指标（涨跌线）确认

### 4.2 成长股评分卡

| 维度 | 权重 | 5分标准 | 3分标准 | 1分标准 |
|------|------|---------|---------|---------|
| 营收增长 | 20% | >30% YoY | 15-30% | <15% |
| 盈利增长 | 20% | >30% YoY | 15-30% | <15% |
| 行业地位 | 15% | 细分第一 | 前三名 | 前十名 |
| 竞争壁垒 | 15% | 极强护城河 | 中等壁垒 | 壁垒较弱 |
| TAM扩张 | 15% | TAM > 10x当前营收 | 5-10x | < 5x |
| 管理层 | 10% | 创始人+优秀记录 | 职业经理人+稳定 | 频繁更换 |
| 估值 | 5% | PEG < 1 | PEG 1-2 | PEG > 2 |

**入选门槛：总分 >= 3.5 / 5.0**

## 五、价值股筛选模型

### 5.1 深度价值筛选标准

**Ben Graham改良版：**
- 市盈率 <= 行业中位数的80%
- 市净率 <= 1.5x（或 PE x PB <= 22.5）
- 股息率 >= 市场平均水平（美股 >= 1.8%，港股 >= 3.5%）
- 资产负债率 <= 50%（非金融）
- 流动比率 >= 1.5
- 连续5年盈利
- 连续5年分红

### 5.2 价值陷阱排除标准

即使满足低估值条件，以下情况需排除或降权：

| 排除信号 | 严重程度 | 说明 |
|----------|---------|------|
| 营收连续3季度下滑 | 高 | 基本面恶化 |
| 行业处于结构性衰退 | 高 | 非周期性低谷 |
| 管理层大规模减持 | 高 | 内部人不看好 |
| 审计报告有保留意见 | 极高 | 财务可信度存疑 |
| 关联方交易占比高 | 高 | 利益输送风险 |
| 应收账款增速远超营收 | 中 | 收入质量存疑 |
| 商誉占总资产 > 30% | 中 | 减值风险 |

### 5.3 价值股评分卡

| 维度 | 权重 | 5分标准 | 3分标准 | 1分标准 |
|------|------|---------|---------|---------|
| 估值折价 | 25% | PE < 0.6x行业 | 0.6-0.8x | 0.8-1.0x |
| 股息安全性 | 20% | 派息率<50%+5年增长 | 派息率<70% | 派息率>70% |
| 资产质量 | 15% | 低杠杆+高现金 | 中等杠杆 | 高杠杆 |
| 盈利稳定性 | 15% | 10年无亏损 | 5年无亏损 | 偶有亏损 |
| 催化剂 | 15% | 有明确重估事件 | 行业回暖 | 无明显催化 |
| 管理层意愿 | 10% | 大手笔回购+增持 | 维持分红 | 无动作 |

**入选门槛：总分 >= 3.5 / 5.0**

## 六、动量股筛选模型

### 6.1 趋势跟踪策略

**入场条件（全部满足）：**
1. 股价 > 200日均线
2. 200日均线斜率向上（近20日）
3. 50日均线 > 200日均线（金叉确认）
4. 近3个月涨幅排名行业前20%
5. 近1个月回调幅度 < 10%（趋势延续而非反转）
6. 成交量确认：上涨日成交量 > 下跌日成交量

**退出条件（任一触发）：**
1. 股价跌破50日均线且3日未收回
2. 50日均线下穿200日均线（死叉）
3. RSI(14) > 80且出现顶背离
4. 单日跌幅 > 5%放量长阴

### 6.2 盈利动量策略

**筛选条件：**
- 最近2个季度EPS均超分析师一致预期 >= 5%
- 分析师EPS上调比例 > 60%（近3个月）
- 分析师目标价上调比例 > 50%
- 公司在最近财报中上调全年指引

**2026年特别关注的盈利动量行业：**
- 半导体（AI芯片需求远超预期）
- 云计算（AI工作负载驱动）
- 数据存储（NAND/HBM价格上行周期）
- GLP-1药物（渗透率加速提升）
- 中国互联网（AI变现初见成效）

### 6.3 动量评分卡

| 维度 | 权重 | 5分标准 | 3分标准 | 1分标准 |
|------|------|---------|---------|---------|
| 价格动量（3M） | 25% | 行业前10% | 前25% | 前50% |
| 价格动量（12M） | 20% | 行业前10% | 前25% | 前50% |
| 盈利超预期 | 20% | 连续4季超预期>10% | 2季超预期 | 偶尔超预期 |
| 分析师修正 | 15% | >80%上调 | >60%上调 | >40%上调 |
| 成交量趋势 | 10% | 放量上涨缩量回调 | 量价基本匹配 | 放量下跌 |
| 机构资金流 | 10% | 大幅净流入 | 小幅净流入 | 净流出 |

**入选门槛：总分 >= 3.8 / 5.0**

## 七、质量因子筛选

### 7.1 高质量企业特征

质量因子是所有策略的基础层，用于确保投资标的具备优异的基本面质量。

**财务质量指标：**

| 指标 | 标准 | 权重 |
|------|------|------|
| ROE稳定性 | 近5年ROE标准差 < 5个百分点 | 15% |
| 毛利率趋势 | 近3年毛利率持平或上升 | 10% |
| 应计利润比率 | < 行业中位数 | 10% |
| FCF/Net Income | > 80%（连续3年） | 15% |
| 资产周转率趋势 | 稳定或上升 | 5% |
| 利息覆盖倍数 | >= 5x | 10% |
| 经营杠杆 | 经营利润增速 > 营收增速 | 10% |

**商业模式质量：**

| 指标 | 标准 | 权重 |
|------|------|------|
| 经常性收入占比 | >= 50% | 10% |
| 客户集中度 | 前5大客户 < 30%营收 | 5% |
| 转换成本 | 高 | 5% |
| 网络效应 | 存在 | 5% |

### 7.2 Piotroski F-Score筛选

约瑟夫·皮奥特罗斯基的九项评分体系用于评估财务健康度：

| # | 指标 | 得1分条件 |
|---|------|----------|
| 1 | 净利润 | > 0 |
| 2 | 经营现金流 | > 0 |
| 3 | ROA变化 | YoY改善 |
| 4 | 应计利润 | OCF > 净利润 |
| 5 | 杠杆变化 | 长期负债/总资产 YoY下降 |
| 6 | 流动性 | 流动比率 YoY改善 |
| 7 | 稀释 | 未发行新股 |
| 8 | 毛利率 | YoY改善 |
| 9 | 资产周转 | YoY改善 |

**评分标准：**
- F-Score >= 7：财务健康，可投资
- F-Score 4-6：需进一步分析
- F-Score <= 3：财务存在隐患，回避

| 频繁更换CFO | 极高 | 3年内更换2次以上 |
| VIE结构风险 | 中 | 中概股特有风险，监管不确定性 |

### 8.3 市场红旗

| 红旗信号 | 严重等级 | 具体表现 |
|----------|---------|---------|
| 极度拥挤交易 | 高 | 做空比率极低 + 估值泡沫化 |
| 叙事驱动型 | 高 | 缺乏盈利支撑的纯概念炒作 |
| 流动性陷阱 | 高 | 日成交量突然枯竭 |
| 被动资金依赖 | 中 | 主要靠指数纳入驱动，基本面不匹配 |
| 杠杆资金集中 | 中 | 融资买入占比过高 |

### 8.4 2026年特别警示

**当前市场环境下需特别警惕的风险：**

1. **AI泡沫风险**：部分AI概念股估值远超基本面支撑，需区分真正受益者与纯概念炒作
2. **特斯拉估值风险**：TTM PE高达~379x，Forward PE ~217x，需靠FSD/Optimus机器人等长期愿景兑现
3. **中美关系不确定性**：贸易政策变化可能影响在美上市的中概股及涉及中国业务的美股
4. **中国房地产尾部风险**：尽管政策托底，万科等开发商仍面临违约风险，产业链公司需谨慎
5. **港股新能源车竞争白热化**：价格战侵蚀利润率，BYD/蔚来/小鹏等需关注毛利率变化
6. **高利率环境余波**：虽然预期降息，但高杠杆企业仍承压
7. **地缘政治风险**：中东局势、俄乌冲突等可能引发能源价格波动

## 九、综合筛选流程

### 9.1 分步筛选法

```
第一层：流动性筛选
  美股：市值 > 20亿美元，日均成交 > 1000万美元
  港股：市值 > 100亿港元，日均成交 > 3000万港元
  ↓ 排除约70%股票

第二层：质量筛选
  F-Score >= 6
  ROE >= 12%
  FCF > 0
  ↓ 排除约60%剩余股票

第三层：策略筛选（三选一）
  A. 成长策略：营收/EPS增速 >= 20%，PEG <= 1.5
  B. 价值策略：PE <= 行业0.8x，股息率 >= 2.5%
  C. 动量策略：3M/12M相对强度前25%，盈利超预期
  ↓ 筛选出约50-100只候选

第四层：红旗排除
  排除存在极高/高严重等级红旗的标的
  ↓ 排除约30%

第五层：深度分析
  逐一进行基本面深度研究
  建立DCF/相对估值模型
  确定目标价位与安全边际
  ↓ 最终观察池：20-40只
```

### 9.2 仓位管理建议

| 分类 | 仓位占比 | 个股上限 | 适用策略 |
|------|---------|---------|---------|
| 核心持仓 | 50-60% | 单只 <= 10% | 高质量成长/GARP |
| 卫星持仓 | 20-30% | 单只 <= 5% | 动量/主题机会 |
| 防御配置 | 10-20% | 单只 <= 5% | 高股息/公用事业 |
| 投机仓位 | 0-10% | 单只 <= 2% | 高风险高回报 |
