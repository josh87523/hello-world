## 关键发现

### 1. 内容类型与数据关系
- **干货帖数据远超日常帖**：搞搞而已干货千赞vs其他50赞以下
- **长篇干货有搜索长尾**：老杨de知识库靠万字拆文积累14个满群
- **连载系列有粘性**：周公何在"挣刀乐第N天"6个满群

### 2. 转化路径
- **低价引流品 → 群聊 → 私域**是主流转化链路
- 问卷/模板类数字产品转化效果好（雨上晴空1.9万+单）
- 群聊是核心转化枢纽，小红书群 → 微信群

### 3. 人设要素
- **"有结果"**：需要数据背书（收入/签约/粉丝量）
- **"活人感"**：口语化、热情、玩梗、评论互动
- **"同人女"**：使用圈内用语，年纪不大的设定

### 4. 封面规律
- 大字报封面点击率高于纯图片
- 浅色底 > 深色底（暗色模式下深色封面不明显）
- 带数据/收入截图的封面点击率高但可能触发平台审核
