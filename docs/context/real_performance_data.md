## 平台审核踩坑记录

1. **收入数据敏感**：封面带收入截图会被风险提示
2. **涨粉引导被禁**："现在关注就是早期云股东""冲刺千粉"等文案不能用
3. **AI标签风险**：文案提到"像AI生成的套话"会被打「AI生成」标
4. **兼职诱导标签**：可能被误判
5. **投稿渠道帖**：流量起来后会被审核ban
6. **违反4.4**：多篇笔记显示违反但客服说状态正常
7. **连赞惩罚**：给大号连赞会被平台警告，暗限约一周
8. **敏感词**：下y、cb、🚗 等需替换（如"拉扯感"）
