# 小红书内容创作指南

> 综合来源：行业研究库 808 篇文章 + 含蓄科技运营实战

## 一、内容系列规划

### 推荐内容系列（经实战验证）

| 系列 | 说明 | 数据表现 | 适合阶段 |
|---|---|---|---|
| 硬干货分享 | 长文/视频，至少2k字。依据选题库 | 互动率高(17-33%)，观看时长翻倍 | 前期起号 |
| 细节干货 | 重选题，戳痛点。如"3个方法快速了解不熟悉的地点/年代/职业" | 中等观看，高收藏 | 前期起号 |
| 热点素材 | 节日/近期热门话题，需提前准备 | 时效性强，错过热点数据差 | 持续更新 |
| 靠XX养活自己的第N天 | 连载系列，自然夹杂干货。人设饱满立体 | 起号效果好，长期更新 | 前期+长期 |
| 圈内吐槽杂谈 | 重互动，引发共鸣讨论 | 互动高但转化低 | 穿插发布 |
| AI相关 | AI写作教程/工具分享 | 时效性要求高 | 按热点穿插 |

### 内容框架模板

**标准图文笔记：**
```
封面：干货主题文字 + 数据图/表情包 + 生活图
正文：
  - 自我介绍（一句话）
  - 引入（钩子/痛点）
  - 干货 1️⃣ 2️⃣ 3️⃣
  - 引导互动转化
评论区：置顶引导
```

## 二、标题法则

### 高点击率标题公式
1. **数字+痛点**："3个方法让你的同人文文笔飙升"
2. **反常识**："新手写小说最快的方式就是抄"
3. **情绪共鸣**："太好了🥹 同人女的文笔有救了"
4. **悬念钩子**："卡文有救了😭新手作者快来试试这招"
5. **自测/挑战**："自测5点，你就是同人圈大佬幼年体🥰"
6. **连载标题**："同人女靠写作养活自己的第N天"

### 标题禁忌
- 不要前置"首先""其次""总之"等AI味连接词
- 避免直接提AI（可能被打标）
- 避免提收入数字（审核敏感）
- 避免涨粉引导语

## 三、封面设计

### 封面类型效果排名（基于实测）
1. **数据截图类**（lofter点赞/评论）：点击率 20-28%
2. **大字报+表情包**：点击率 15-22%
3. **长文自带封面**：点击率 13-14%，但互动率最高
4. **实拍图**：点击率 13-15%
5. **纯黑底海报**：点击率 5-14%（暗色模式不友好）

### 封面最佳实践
- 比例 3:4（小红书标准）
- 浅色底 > 深色底
- 大字突出选题关键词
- 加一个情绪元素（表情包/数据图）
- 避免放收入数据（触发审核）

### 封面测试方法
1. **AB测试**：同一选题间隔3天发两版
2. **四象限定位**：横轴=信息浓度，纵轴=情绪价值
3. **赛马机制**：5个封面各投30元薯条，6小时看数据
4. **灵感来源**：Pinterest/Instagram搜封面 → 灰豚/新红看数据 → 稿定设计/可画做模板

## 四、正文写作规范

### 结构要求
- **开头3行定生死**（小红书展开前只显示3行）
- 用一句话钩子抓注意力：反常识/共鸣/悬念
- 中间干货段落用emoji分隔，重点加粗
- 结尾引导互动（提问/投票/号召）

### 字数指引
- 标准笔记：800-1500字（黄金长度）
- 短篇攻略：1000字左右
- 深度干货：2000字以上

### 去AI味要素（重要！）
- 删除"首先""其次""总之""综上所述"等连接词
- 避免完美的排比句式
- 加入口语化表达、语气词
- 插入个人经历/感受
- 适当的不完美（错别字、省略号、语气断裂）
- 使用圈内专属用语/缩写
- 避免文案中提到"AI"相关词汇（可能被打标）

### 语气调性
- 像朋友分享经验，不要官方说教
- 口语化、热情、玩梗
- 有情绪起伏（开心/吐槽/崩溃都可以）
- "活人感"是核心

## 五、标签策略

- 5-10个标签最佳
- 格式统一加#
- 混合大标签（流量大竞争大）和小标签（精准流量）
- 关键词布局要考虑搜索长尾

## 六、发布时间

### 最佳发布时间（实测）
- ❌ 18:00 偏冷
- ✅ 20:00 - 20:30 效果最好
- 备选：07:00-09:00、12:00-14:00

### 发布后动作
- 发布后30分钟内回复评论有助于推荐
- 评论区置顶引导
- 前期要积极互动（特别是评论）

## 七、数据复盘指标

| 指标 | 计算方式 | 对应优化项 |
|---|---|---|
| 观看率 | 观看量/曝光量 | 内容形式（图文/视频） |
| 点击率 | 点击量/曝光量 | 封面/标题 |
| 互动率 | 互动量/观看量 | 选题内容质量 |
| 涨粉率 | 涨粉数/观看量 | 选题+人设引导 |
| 进群比 | 新增进群/观看量 | 钩子吸引力 |
| 转化率 | 微信群新增/小红书群新增 | 群内引导 |

## 八、方法论总结（梅常书12条军规摘要）

1. 小红书本质是「数字版生活杂志」
2. 深耕垂类领域的深度内容
3. 持续产出「看完就能用」的优质笔记
4. 精准关键词布局（搜索流量 > 推荐流量的长期价值）
5. 流量分发机制与抖音显著不同（内容质量 > 账号权重）
6. 冷启动阶段特别提示：好内容会被推荐，不依赖粉丝基数
7. 矩阵式内容规划
8. 搜索流量优化（长尾关键词）
9. 违规红线规避（参见平台审核踩坑记录）
